# FIFA 2026

A web-based project dedicated to FIFA World Cup 2026 content and information.

## 📋 Project Overview

This repository contains a static web application built with HTML, JavaScript, and CSS to provide information, updates, and content related to the FIFA World Cup 2026 tournament.

## 🚀 Getting Started

### Prerequisites

No installation required! This is a static website that can be run directly in any modern web browser.

### Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/chealseainfo1/fifa2026.git
   cd fifa2026
   ```

2. Open `index.html` in your web browser:
   - Double-click the file, or
   - Use a local server (recommended):
     ```bash
     python -m http.server 8000
     ```
   - Then navigate to `http://localhost:8000` in your browser
